# Academic Attendance & Reporting Web System

A **full-stack web application** designed to manage student attendance efficiently for educational institutions. The system enables administrators and faculty members to manage student records, mark attendance in real time, generate reports, and send automated email notifications through a secure and responsive interface.

This project is developed **from scratch** using core web technologies, following clean coding practices and real-world application architecture.

---

##  Features

- Secure user authentication (Admin & Faculty roles)
- Student and class management
- Real-time attendance marking using AJAX (no page reload)
- Attendance reports and analytics
- Automated email notifications
- Responsive and user-friendly UI
- Secure database operations with MySQL

---

##  Tech Stack

- **Backend:** PHP  
- **Database:** MySQL  
- **Frontend:** HTML, CSS, JavaScript  
- **Asynchronous Requests:** AJAX  
- **Email Service:** SMTP / PHP Mail  
- **Version Control:** Git & GitHub
